ls - listar arquivos e pastas dentro de um diretorio

cd - é para movimentação entre diretorios

git status - mostra tudo que foi modificado no seu repositorio local

git add . para adicionar todos os arquivos a enviar para commit

git commit -m "escrever o texto referencia do commit"

git push origin (espaço mais o nome da brach na frente)

git pull origin (espaço mais o nome da brach na frente)

git branch - listar todas as branchs do repositório local

git fetch - trazer todo o historico da branch

git switch (espaço mais o nome da brach na frente) - mudar para a brach desejada

git checkout (espaço mais o nome da brach na frente)- criar a branch

git checkout -b (espaço mais o nome da brach na frente)- criar a branch

git branch -d  (espaço mais o nome da brach na frente)- deletar a branch
